Place ici tes fichiers audio (.wav ou .mp3) pour activer les effets sonores.

Noms attendus par le jeu (tous optionnels) :

    night.wav     - ambiance "nuit qui tombe" (hibou, chouette)
    day.wav       - cloche du village au lever du jour
    howl.wav      - hurlement de loup
    death.wav     - mort d'un joueur
    victory.wav   - fin de partie

Si un fichier n'existe pas, le jeu se contente de l'ignorer.
