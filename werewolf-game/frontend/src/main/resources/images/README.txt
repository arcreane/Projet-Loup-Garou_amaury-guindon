==============================================================
IMAGES - À TÉLÉCHARGER (optionnel)
==============================================================

Le jeu fonctionne SANS images (fallback CSS pur), mais il sera
beaucoup plus immersif avec les assets ci-dessous.

Tout est gratuit, sources : Pixabay, Unsplash, OpenGameArt.
