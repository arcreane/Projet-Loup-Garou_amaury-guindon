==============================================================
POLICES D'ÉCRITURE - À TÉLÉCHARGER
==============================================================

Place ici les fichiers .ttf suivants pour activer le style gothique.
Tous gratuits, sous licence Open Font License (OFL).

1) Cinzel-Bold.ttf
   Titres principaux. Style romain/gothique majestueux.
   https://fonts.google.com/specimen/Cinzel
   → Onglet "Download family" → décompresse → prends "Cinzel-Bold.ttf"

2) IMFellEnglish-Regular.ttf
   Accents thématiques (sous-titres, badges).
   https://fonts.google.com/specimen/IM+Fell+English
   → "Download family" → "IMFellEnglish-Regular.ttf"

3) IMFellEnglish-Italic.ttf
   Variante italique pour citations / pensées.

==============================================================

Si une police est absente, le jeu utilisera Segoe UI en fallback
sans crash. Le style sera juste moins thématique.

Pour vérifier que ça marche : lance le jeu, l'écran de connexion
doit afficher "Loup-Garou" en grandes lettres romaines.
